# Archiving tools and instructions

These are some of the tools we're using to capture cultural heritage sites.

* [Archiving URLs with Internet Archive](https://help.archive.org/help/save-pages-in-the-wayback-machine/)
    * [Batch processing URLs with Google Sheets and Internet Archive](https://www.sucho.org/ia-gsheets)
    * [Programatically bulk uploading to the Internet Archive](https://www.sucho.org/ia-bulk-upload)
* [Browsertrix](https://www.sucho.org/browsertrix)
    * [Browsertrix Cloud](https://www.sucho.org/browsertrix-cloud)
* [Proof of concept for embedding web archives with the ReplayWeb.page web component](https://inkdroid.org/web-archives)
* [Webrecorder](https://webrecorder.net/)
    * [Webrecorder Browser plugin](https://www.sucho.org/webrecorder-browser-plugin-instructions)
    * [ArchiveWebPage app](https://www.sucho.org/archivewebpage-app-instructions)
